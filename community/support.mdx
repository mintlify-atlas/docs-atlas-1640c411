## How to Report Issues

If you encounter problems with any of the llms.txt files, please help us improve by reporting them:

### For Documentation Issues

1. **Check existing issues** - Search [GitHub issues](https://github.com/sin4ch/nigerian-fintech-llms-txt/issues) to see if it's already reported
2. **Open a new issue** - Include:
   - Which provider's llms.txt file has the issue
   - What's incorrect or missing
   - Link to the official documentation (if applicable)
   - How it affects your AI tool's understanding

### For Integration Problems

If you're having trouble using these files with your AI tools:

1. Verify your AI tool supports the llms.txt standard
2. Check that you're pointing to the correct file path
3. Share your use case in [GitHub Discussions](https://github.com/sin4ch/nigerian-fintech-llms-txt/discussions)
