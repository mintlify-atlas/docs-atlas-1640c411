## Overview

Paystack is Africa's leading payments infrastructure provider, enabling businesses to accept payments from customers via cards, bank accounts, mobile money, USSD, QR codes, and bank transfers. The platform operates in **Nigeria, Ghana, South Africa, Kenya, and Côte d'Ivoire**, supporting both online and in-person payments.

## Integration Options

<Accordion title="Popup JS (InlineJS)">
  JavaScript SDK for web payment integration with Popup checkout. Best for web applications requiring seamless in-page checkout.
  
  ```javascript
  const paystack = new PaystackPop();
  paystack.newTransaction({
    key: 'pk_test_xxxxxxxxxxxxx',
    email: 'customer@email.com',
    amount: 10000, // Amount in kobo (100 NGN)
    onSuccess: (transaction) => {
      console.log('Payment successful!', transaction);
    },
    onCancel: () => {
      console.log('Payment cancelled');
    }
  });
  ```
  
  [View Documentation](https://paystack.com/docs/developer-tools/inlinejs/)
</Accordion>

## Identity Verification

Paystack provides KYC tools for verifying customer identity and bank details:

- **Verify Account Number**: Confirm bank account ownership before transfers (Nigeria & Ghana)
- **Validate Customer**: BVN and identity verification for Dedicated Virtual Account customers
- **Resolve Card BIN**: Get card information from the first 6 digits of a card number

[Learn more about Identity Verification](https://paystack.com/docs/identity-verification/)

## Important Notes

<Note>
  - All amounts are in **smallest currency unit** (kobo for NGN, pesewas for GHS, cents for KES/ZAR/USD)
  - Always **verify transactions server-side** before delivering value
  - Test mode uses test API keys (`sk_test_*`, `pk_test_*`)
  - Live mode uses live keys (`sk_live_*`, `pk_live_*`)
  - Webhooks provide real-time notifications; validate signatures using your secret key
</Note>
